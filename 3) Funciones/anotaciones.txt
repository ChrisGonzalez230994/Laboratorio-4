üîπJSON.stringify()

Convierte un objeto de JavaScript a una cadena JSON (texto).

Se usa cuando quieres guardar, enviar o almacenar datos (por ejemplo, en un archivo, en una base de datos o al enviarlos a un servidor).

Ejemplo:

const persona = { nombre: "Ana", edad: 25 };
const json = JSON.stringify(persona);

console.log(json); 
// Resultado: '{"nombre":"Ana","edad":25}'  (string, no objeto)

üîπ JSON.parse()

Convierte una cadena JSON a un objeto de JavaScript.

Se usa cuando recibes datos en formato JSON (por ejemplo, desde una API o archivo) y necesitas trabajar con ellos en tu c√≥digo.

Ejemplo:

const json = '{"nombre":"Ana","edad":25}';
const persona = JSON.parse(json);

console.log(persona.nombre); // Resultado: Ana
console.log(persona.edad);   // Resultado: 25
